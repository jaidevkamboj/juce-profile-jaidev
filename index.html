Course Curriculum

Module 01
🛠️

JUCE Environment Setup & Architecture

Complete C++ audio plugin development environment and JUCE project structure.

  • Installing JUCE & CMake on Windows/macOS
  • Projucer vs CMake build systems
  • JUCE module system & project anatomy
  • Configuring VST3, AU, AAX targets
  • Plugin vs standalone application mode
6 lessons ~3 hrs
Module 02
⚙️

AudioProcessor Deep Dive

Master the core AudioProcessor class — the engine of every JUCE plugin.

  • prepareToPlay / processBlock lifecycle
  • AudioBuffer manipulation & channel routing
  • APVTS: parameter management system
  • State save/restore (getStateInformation)
  • Thread safety & real-time constraints
7 lessons ~4 hrs
Module 03
🎛️

DSP Fundamentals for Audio Programmers

The digital signal processing math that powers every audio effect.

  • Samples, sample rates & Nyquist theorem
  • Gain, decibels & amplitude math
  • FIR vs IIR filters – theory to implementation
  • Convolution & impulse responses
  • Fast Fourier Transform (FFT) basics
8 lessons ~4.5 hrs
Module 04
🎵

Building Audio Effects – EQ, Compressor & Delay

Build three complete professional-grade audio effects from scratch.

  • Parametric EQ with multiple bands
  • Dynamic range compressor with attack/release
  • Stereo delay with feedback & mix control
  • Parameter smoothing for artifact-free automation
  • Testing plugins in a DAW (Reaper, Ableton)
9 lessons ~5 hrs
Module 05
🎹

MIDI Processing & Synthesizer Development

Turn your plugin into a fully functional MIDI synthesizer.

  • MIDI message handling & note events
  • Oscillator design: sine, saw, square, wavetable
  • ADSR envelope implementation
  • Voice stealing & polyphony management
  • Building a complete subtractive synth
8 lessons ~5 hrs
Module 06
🖥️

Custom GUI Design with JUCE Components

Design professional, visually polished plugin user interfaces.

  • AudioProcessorEditor architecture
  • Custom sliders, buttons & combo boxes
  • OpenGL rendering for high-performance graphics
  • Look & Feel customization system
  • Responsive layouts & plugin resizing
7 lessons ~4 hrs

Spectrum Analyzer & Visualizations

Build real-time audio visualizations — spectrum analyzers, oscilloscopes, meters.

  • FFT-based spectrum analysis implementation
  • Real-time waveform oscilloscope
  • Peak & RMS level meter components
  • FIFO audio data passing to GUI thread
  • Smooth animation with juce::Timer
6 lessons ~3.5 hrs

Plugin Formats, Packaging & Distribution

Compile, test, and distribute your plugins across all major formats.
